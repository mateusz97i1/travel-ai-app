# 🌍 TravelAI

**TravelAI** is an intelligent travel planning assistant designed to make dream vacations simple, exciting, and stress-free. By leveraging smart algorithms, the app transforms the way users plan their journeys, moving from hours of manual research to instant, personalized itineraries.

## 🛠️ Tech Stack

* **Backend:** [Django](https://www.djangoproject.com/) (Python)
* **Frontend:** [Tailwind CSS](https://tailwindcss.com/)
* **Template Engine:** Django Templates (utilizing Template Inheritance and Custom Blocks)
* **Styling:** Gradient buttons, Shadow-sm cards, and Responsive Grid layouts.

---

## 📋 How It Works

1.  **Select Destination:** Choose where you want to go.
2.  **Set Duration:** Tell the AI how many days you have available.
3.  **Generate:** Click the **"Generate plan"** button to receive a curated, step-by-step itinerary optimized for your needs.

## 🎨 UI Highlights

- **Custom Banners:** Features a dedicated `{% block banner %}` for flexible content injection.
- **Card-Based Info:** Uses a 3-column grid to highlight core value propositions (Optimization, Effortless Planning, and Experiences).
- **Interactive CTA:** Includes a prominent "Free try!" call-to-action with a custom-styled gradient button.
